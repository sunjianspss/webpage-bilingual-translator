# v0.1.3 - Initial Open Source Release

This is the first public release of зҪ‘йЎөеҸҢиҜӯзҝ»иҜ‘, a bilingual webpage translator extension for Chrome, Edge, and Safari.

## Highlights

- Chrome / Edge Manifest V3 extension that can be loaded directly from the repository root.
- Safari Web Extension project with a native proxy for local API requests.
- Supports local OpenAI-compatible APIs such as LM Studio and Ollama.
- Supports DeepSeek API with Flash and Pro model options.
- Full-page visible-text translation with bilingual and translated-only display modes.
- Restore-original action and page-level progress/error feedback.
- X.com dynamic-page handling with visible-content rescans.
- Structured social-post translation that preserves paragraph breaks and list-like layout.
- Local API optimizations: compact request payloads, dynamic token budget, fast first batch, and controlled worker concurrency.
- Shortcut command support for translating the current page.

## Installation

### Chrome / Edge

1. Download the source code from this release and unzip it.
2. Open `chrome://extensions/` or `edge://extensions/`.
3. Enable Developer mode.
4. Click "Load unpacked".
5. Select the unzipped project root.

### Safari

Safari support is included as an Xcode project under `safari/зҪ‘йЎөеҸҢиҜӯзҝ»иҜ‘/`.

For local development:

```bash
./script/build_and_run.sh
```

For long-term everyday use, Safari requires signing with an Apple Developer certificate.

## Configuration

Default local API settings:

```text
API URL: http://127.0.0.1:1234/v1
Model: qwen/qwen3.5-35b-a3b
```

The extension works with OpenAI-compatible `POST /chat/completions` services. DeepSeek users should select "DeepSeek API" in the popup and enter their API key locally in the browser extension.

## Known Limits

- The default page cap is 220 unique visible text segments.
- Newly loaded content after scrolling can be translated by clicking the translate action again.
- Complex site CSS may still cause small layout differences.
- API keys are stored in browser extension local storage, which is suitable for personal use. Team deployments should use a backend proxy.
